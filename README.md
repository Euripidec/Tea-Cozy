# The Tea Cozy ☕

A responsive landing page for a fictional tea shop, built as a front-end practice project. Features a fixed navigation header, a mission section with a background image, a featured tea gallery, store locations, and a contact footer.

## Preview

![The Tea Cozy](https://content.codecademy.com/courses/freelance-1/unit-4/img-tea-cozy-logo.png)

## Features

- **Fixed navbar** with smooth anchor-link navigation
- **Mission section** with a full-width background image
- **Tea of the Month** gallery with a responsive flexbox grid
- **Locations section** with a background image and address cards
- **Contact & Footer** with shop info

## Project Structure

```
tea-cozy/
├── index.html    # Page structure and content
└── style.css     # Layout and styling
```

## Running Locally

No dependencies required. Clone the repo and open `index.html` directly in your browser.

```bash
git clone https://github.com/Euripidec/tea-cozy.git
cd tea-cozy
open index.html
```

## Built With

- HTML5
- CSS3 (Flexbox, fixed positioning, background images)
- Helvetica / sans-serif system fonts
